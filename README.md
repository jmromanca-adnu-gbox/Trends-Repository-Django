How to run:
1. Clone the repository into your local drive.
2. Create a virtual Python Environment.
3. Open the Terminal and navigate to the location of the repository.
4. Run the command `python manage.py runserver` and Ctrl+Click on the address of `http://127.0.0.1:8000/`
5. Change URL in the address bar of the browser to `http://127.0.0.1:8000/members/`
